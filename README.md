## Install

```sh
curl -o- https://war10ck.thomaslaurenson.com/install.sh | bash
```

## What?!

This is a ridiculously opinionated tool to install and configure Debian/Ubuntu based operating systems. All of the modules are specifically authored for me and how I like my system configured. Only ever used or tested on Debian 13. However, you could fork and edit or replace modules for yourself and use the underlying functionality such as the installer and library.
